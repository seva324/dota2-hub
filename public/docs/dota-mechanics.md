# Dota 2 游戏机制详解

> 本文档整理自 Liquipedia 和 Dota 2 Wiki，系统性地介绍 Dota 2 的核心机制。

## 一、分路 (Lanes)

### 1.1 三路分布

Dota 2 有三条主要路线：

| 分路 | 天辉 (Radiant) | 夜魇 (Dire) | 特点 |
|------|---------------|-------------|------|
| 上路 (Top) | 优势路 | 劣势路 | 靠近赏金符点 |
| 中路 (Mid) | 中路 | 中路 | 最短路径，经验最高 |
| 下路 (Bot) | 劣势路 | 优势路 | 靠近远古野区 |

**关键概念**：
- **优势路 (Safe Lane)**：己方防御塔更近，安全性更高，兵线更容易控
- **劣势路 (Off Lane)**：敌方防御塔更近，容易被压制，但可以干扰敌方发育

### 1.2 分路编号 (Lane ID)

OpenDota API 中的 `lane` 字段：
- `lane = 1`：优势路（从己方队伍视角）
- `lane = 2`：中路
- `lane = 3`：劣势路（从己方队伍视角）

### 1.3 分路角色 (Lane Role)

OpenDota API 中的 `lane_role` 字段：
- `lane_role = 1`：一号位 (Carry)
- `lane_role = 2`：二号位 (Mid)
- `lane_role = 3`：三号位 (Offlane)
- `lane_role = 4`：四号位 (Soft Support)
- `lane_role = 5`：五号位 (Hard Support)

## 二、兵线机制 (Creep Mechanics)

### 2.1 兵线类型

| 类型 | 生命值 | 护甲 | 攻击力 | 金币 | 经验 |
|------|--------|------|--------|------|------|
| 近战兵 (Melee) | 550 | 2 | 19-23 | 34-39 | 57 |
| 远程兵 (Ranged) | 300 | 0 | 21-26 | 43-52 | 69 |
| 炮车 (Siege) | 935 | 0 | 35-46 | 59-72 | 88 |
| 旗手 (Flagbearer) | 550 | 2 | 19-23 | 34-39 | 57 |

### 2.2 刷兵时间

- **每30秒**生成一波兵
- **每分钟**（第5波起）生成旗手，替代一个近战兵
- **每5分钟**（第11波起）生成炮车

### 2.3 兵线进化

| 时间 | 兵线变化 |
|------|----------|
| 0:00 | 初始：3近战 + 1远程 |
| 15:00 | 全路线 +1 近战兵 |
| 30:00 | 全路线再 +1 近战兵 |
| 35:00 | +1 炮车 |
| 40:00 | +1 远程兵 |
| 45:00 | +1 近战兵 |

### 2.4 超级兵 (Super Creeps)

- 摧毁任意一路兵营后，该路开始刷超级兵
- 摧毁所有兵营后，全地图刷超级兵

### 2.5 兵线控制技巧

1. **拉野 (Pulling)**：拉野怪打兵线，干扰敌方发育
2. **囤野 (Stacking)**：同时拉多波野怪，增加刷野效率
3. **控线 (Creep Equilibrium)**：保持兵线在理想位置
4. **推线 (Pushing)**：快速推塔
5. **勾兵 (Aggro)**：通过攻击敌方英雄吸引小兵仇恨

### 2.6 仇恨机制 (Aggro)

- 近战兵仇恨范围：500
- 远程兵仇恨范围：600
- 炮车仇恨范围：800
- **5分钟前**小兵无法被玩家单位仇恨（除非在塔下1500范围内）

## 三、经济系统 (Gold Mechanics)

### 3.1 两种金币

| 类型 | 来源 | 用途 |
|------|------|------|
| 可靠金币 (Reliable) | 被动收入、赏金符 | 买活优先消耗 |
| 不可靠金币 (Unreliable) | 击杀、推塔、助攻 | 购买装备 |

### 3.2 击杀奖励

**英雄击杀**：
- First Blood：135金币 + 100基础经验
- 击杀经验公式：`100 + 0.13 × 死亡英雄经验值`
- 助攻金币公式：`(30 + 0.038 × 死亡英雄净经济) / 助攻人数`

**连杀奖励 (Kill Streak)**：
| 连杀数 | 名称 | 金币奖励 |
|--------|------|----------|
| 3 | Spree | 60 |
| 4 | Dominating | 100 |
| 5 | Mega Kill | 150 |
| 6 | Unstoppable | 210 |
| 7 | Wicked Sick | 280 |
| 8 | Monster Kill | 360 |
| 9 | Godlike | 450 |
| 10+ | Beyond Godlike | 550 |

### 3.3 被动收入

- 初始：90金币/分钟
- 随时间递增（每分钟+0.8左右）
- 45分钟时约105金币/分钟

### 3.4 建筑奖励

| 建筑 | 团队金币 | 最后一击额外 |
|------|----------|-------------|
| 一塔 | 90 | +110~130 |
| 二塔 | 110 | +130~150 |
| 三塔 | 125 | +150~170 |
| 四塔 | 145 | +170~190 |
| 近战兵营 | 155 | +90~135 |
| 远程兵营 | 90 | +90~135 |

## 四、经验系统 (Experience Mechanics)

### 4.1 经验获取

- **经验范围**：1500
- **分配方式**：范围内所有友方英雄平分
- **死亡英雄必须满血后才能获得经验**

### 4.2 升级经验表

| 等级 | 累计经验 | 升级所需 |
|------|----------|----------|
| 1→2 | 0→240 | 240 |
| 2→3 | 240→640 | 400 |
| 3→4 | 640→1160 | 520 |
| 4→5 | 1160→1760 | 600 |
| 6 (大招) | 2440 | 760 |
| 25 (满级) | 34400 | 4000 |

最高等级30级，需63900经验值。

### 4.3 经验来源

**野怪经验**：
| 野怪 | 经验 |
|------|------|
| 小野 | 14-30 |
| 大野 | 40-90 |
| 远古 | 95-124 |
| 肉山 | 大量+重生加速 |

### 4.4 击杀经验

- 英雄死亡基础经验：100
- 每7.69点当前经验额外+1点经验奖励
- 助攻经验与击杀者相同

## 五、地图机制

### 5.1 防御塔 (Towers)

| 塔位置 | 生命值 | 护甲 | 攻击力 |
|--------|--------|------|--------|
| 一塔 | 1600 | 14 | 100-112 |
| 二塔 | 1800 | 16 | 112-125 |
| 三塔 | 2000 | 18 | 125-138 |
| 四塔(高地) | 2200 | 20 | 140-155 |

**防御塔机制**：
- 攻击速度：1秒/次
- 仇恨范围：700
- 背塔保护：5分钟内小兵无法攻击防御塔

### 5.2 肉山 (Roshan)

- 首次击杀：经验 + 掉落不朽盾
- 每次重生：经验+50%，最大+400%
- 经验公式：`基础 + 0.5 × (游戏分钟 / 0.375)`

### 5.3 符点 (Runes)

| 符 | 效果 |
|-----|------|
| 赏金符 | 团队金币 |
| 加速符 | 移动速度+30% |
| 隐身符 | 隐形 |
| 双倍伤害符 | 攻击力翻倍 |
| 幻象符 | 幻象 |
| 恢复符 | 生命/魔法恢复 |
| 智慧符 | 经验加成 |

## 七、数据字段参考

### OpenDota API 关键字段

```typescript
interface Player {
  player_slot: number;     // 0-127 = 天辉, 128-255 = 夜魇
  lane: number;             // 1=优势路, 2=中路, 3=劣势路
  lane_role: number;        // 1-5 号位
  gold_t: number[];         // 时间序列金币
  xp_t: number[];          // 时间序列经验
  lh_t: number[];          // 时间序列正补
  dn_t: number[];          // 时间序列反补
  lane_kills: number;       // 线杀
  lane_efficiency: number;  // 线优率 (0-1)
}
```
